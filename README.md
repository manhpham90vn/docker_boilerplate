# My dotfiles

## link
```shell
brew link --force --overwrite node@20
```

## start service
```shell
brew services start postgresql@16
```

## info serivce
```shell
brew services info postgresql@16
```

## restart serivce
```shell
brew services restart nginx
```

## list casks
```shell
brew list --cask
```

## list formula
```shell
brew leaves
```

## auto remove
```shell
brew autoremove
```
