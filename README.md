# My dotfiles

## bootstrap
```shell
./bootstrap
```

## unlink
```shell
brew unlink node@16
```

## link
```shell
brew link --force --overwrite node@20
```

## start service
```shell
brew services start postgresql@16
```

## info serivce
```shell
brew services info postgresql@16
```

## restart serivce
```shell
brew services restart nginx
```
